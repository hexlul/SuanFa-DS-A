#算法与数据结构
算法是“解决问题的程序”，必须具备两个条件：有效性和终止性。“断点”是表示有效性的方法之一。 

##重要的算法种类：
1.技术计算：欧几里得相除法求最大公约数，埃拉托色尼筛法求素数…… 2.排序：选择排序，插入排序，冒泡排序，快速排序，希尔排序，归并排序，堆排序…… 
3.查找：线性查找，二分查找…… 
4.字母串模式匹配：KMP法，BM法……
